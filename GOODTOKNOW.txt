in geval je de commando's vergeten bent: 
===============================================================
git clone "url" -> voor de map op te laden
cd PATHTOFOLDER -> naar de juiste map gaan

git status -> status van uw veranderde mappen zien 
git add . -> de aangepaste dingen toevoegen in rij 
git commit -m "message" -> in message zetten wat je hebt aangepast 
git push -> de aangepaste dingen uploaden in github
